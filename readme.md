## vshell
vshell is unix shell emulator. This is CLI shell application for a work in virtual filesystems in .tar and .zip files.

Supported commands:
* ls (-a, -l flags)
<br>
-a - show hidden files
<br>
-l - list view 
* pwd
* cd
* cat
* echo

Program support script files by inside archive.

### Usage

$ python3 main.py --archive [path to archive] --script [path to script inside archive]
